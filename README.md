# Easel

Easel is a HTML5 widget that allows drawing and writing on HTML5 canvas elements. Mobile devices are fully supported. 

It is a customizable WYSIWYG HTML canvas editor for freehand drawing and creating sketches with simple shapes.

#### FORK
This is a fork of the original [Drawerjs](https://github.com/carstenschaefer/DrawerJs) project. The fork aims to provide better maintainability, modern dependencies, and additional features. And added a React wrapper and webpack support.
